# Sales-Trend-Analysis-SQL
## Objective
SQL project analyzing monthly sales trends using aggregate functions (SUM, COUNT, GROUP BY, ORDER BY) to calculate revenue and order volume with MySQL.

## Tools
- MySQL Workbench
- SQL

## Dataset
Orders dataset containing:
- order_id
- order_date
- amount
- product_id

## SQL Concepts Used
- GROUP BY
- ORDER BY
- SUM()
- COUNT(DISTINCT)
- YEAR()
- MONTH()

## Files Included
- Sales_Analysis
- README.md
- Screenshots

## Outcome
Successfully analyzed monthly revenue and order volume using SQL aggregations.
